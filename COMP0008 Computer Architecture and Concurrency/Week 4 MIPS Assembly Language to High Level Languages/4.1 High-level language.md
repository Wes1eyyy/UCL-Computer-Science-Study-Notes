## ğŸ“š How to Use the Stack in MIPS

### ğŸ“¦ What is the Stack?

- A **last-in, first-out (LIFO)** memory region
    
- Grows **downwards**
    
- Used to save:
    
    - Return addresses (`$ra`)
        
    - Local variables
        
    - Saved registers (`$s0-$s7`)
        

---

### ğŸ“ Stack Pointer

- **`$sp`** points to the **top of the stack**
    
- Default: high memory (e.g., `0x7ffffffc`)
    

---

## ğŸ”½ Push to Stack


---

## ğŸ”¼ Pop from Stack
